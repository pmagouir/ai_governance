Data Verification Tool

Internal tool for testing data pipeline endpoints and submitting verification check-ins

Endpoint: https://script.google.com/macros/s/AKfycbx967okB59zHn8xmN3TdOfhxg7EmP0lT6ta7rH_4wBKnno_Zi5iKAjyeziRMDiWNyz9Nw/exec

1. Test Endpoint Connectivity

Sends a GET request to verify the endpoint is responding.

2. Send Test Weekly Check-In

Submits a test weekly check-in with name "[VERIFICATION TEST]" and week 0. Attempts CORS-enabled request first, then falls back to no-CORS mode.

3. Send Test Milestone

Submits a test milestone submission for verification. Attempts CORS-enabled request first, then falls back to no-CORS mode.
